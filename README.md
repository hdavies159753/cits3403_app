# cits3403_app
CITS3403 Agile Web Dev Project

# Steps for running flask server in testing
1. Have python installed
2. python -m venv venv
3. venv\scripts\activate (you might have to update your execution policy with Set-ExecuitonPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned)
4. pip install -r requirements.txt
5. python run.py
6. Navigate to http://127.0.0.1:5000
